===========================
How to Use 1-my_list.py
===========================

The module defines a class ``MyList`` that inherits from list. The class defines a method ``print_sorted()``
that prints the list in sorted ascending order

Usage
=====

Import module:
::

	>>> MyList = __import__('1-my_list').MyList

Now use it:
::

	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(4)
	>>> print(my_list)
	[1, 4, 2, 3, 4]

	>>> my_list.print_sorted()
	[1, 2, 3, 4, 4]

	>>> print(my_list)	
	[1, 4, 2, 3, 4]
